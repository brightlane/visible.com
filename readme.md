# Visible Wireless USA

**Live site:** https://brightlane.github.io/visible.com/

USA affiliate site for Visible wireless — $20/month unlimited 5G on Verizon's network. Targets phone plan, prepaid wireless, and Verizon alternative keywords across all 50 states, 238 major US cities, 15 plan types, 20 competitors, and 250 long-tail keywords.

## What this site does

Generates 13,000+ SEO pages covering every wireless phone plan search angle in the USA. Every page includes a price badge, 4 feature cards, FAQ section, and related plan links.

- **State pages** — "Visible unlimited plan Texas", "Visible 5G coverage California", "Visible prepaid plan Florida" — 50 states × 40 intents = 2,000 pages
- **City pages** — "Visible cheap phone plan Houston TX", "Visible no contract plan Chicago IL" — 238 cities × 40 intents = 9,520 pages
- **Plan pages** — dedicated pages for each plan type × 20 intents + overview = 315 pages
- **State × Plan combos** — "Visible Unlimited Plan in Texas", "Visible Senior Plan in Florida" — 50 × 15 = 750 pages
- **Competitor comparisons** — "Visible vs AT&T", "Visible vs T-Mobile", "Visible vs Cricket" × 12 angles = 240 pages
- **Long-tail pages** — 250 research and informational pages

## How it works

`build.py` runs daily via GitHub Actions and generates all HTML pages, `sitemap.xml`, `robots.txt`, and `llms.txt` from scratch, then commits and pushes to main. GitHub Pages deploys automatically on every push.

## Files

| File | Purpose |
|---|---|
| `build.py` | Main page generator — run this to rebuild the site |
| `.github/workflows/visible-engine.yml` | Daily build trigger (2 AM UTC) |
| `.github/workflows/pages.yml` | GitHub Pages deployment |
| `index.html` | Homepage — generated by build.py |
| `sitemap.xml` | Full sitemap — generated by build.py |
| `robots.txt` | Crawl rules — generated by build.py |
| `llms.txt` | LLM-readable site summary |
